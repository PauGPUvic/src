# Match
 A tiny game in CL

## How to compile?
JDK is required to compile
Compile your java classes
```bash
    javac *.java
```
Create a manifest file and your jar file:
```bash
    jar cvfe name.jar name *.class
```
Test your jar:
```bash
    name.jar
```
