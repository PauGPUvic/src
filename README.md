# Match
 A tiny game in CL

## How to compile?
1. Compile your java classes
```bash
    javac *.java
```
2. Create a manifest file and your jar file:
```bash
    jar cvfe name.jar name *.class
```
3.Test your jar:
```bash
    name.jar
```
